### Project Overview

The primary objective of this project, titled 'Challenge', is to implement a chat-based application that leverages AI to create dynamic and interactive exercises designed to help users manage emotions such as anger, anxiety, and sadness. It utilizes technologies like Node.js, Express, Langchain.js, and OpenAI to build a responsive and engaging user experience.

The project's main functionalities include:
- Providing a set of predefined activities aimed at emotional management
- Leveraging AI models to generate responses based on user inputs
- A structured and expandable code architecture for future enhancements

### Installation

1. Clone this repository: `git clone https://github.com/tu-usuario/tu-proyecto.git`
2. Install the dependencies: `npm install`

### Usage

1. Configure the environment variables in the `.env` file.
2. Start the application: `npm run dev`
3. Access the application in your browser: `http://localhost:<port>`

### Folder Structure

- `/app`: Main logic of the application
  - `/controllers`: Application controllers
  - `/models`: Application models
  - `/services`: Application services
- `/config`: Application configuration files
- `/public`: Static files for the application
- `/routes`: Routing files of the application
- `/views`: Application views
- `/middlewares`: Application middlewares
- `/utils`: Utility functions for the application
- `/ai`: AI-related files
  - `/chat`: Files related to chat integration

### Technologies Used

- Node.js
- Express
- Langchain.js
- OpenAI
