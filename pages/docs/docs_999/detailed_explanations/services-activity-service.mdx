### Activity Service

The activityService.js file handles the logic for obtaining the list of available activities by interacting with the activity model.

**Example:**

```javascript
const activityModel = require('../models/activityModel');

module.exports = {
  getActivity: () => {
    return activityModel.getActivity();
  },
};
```

### Chat Service

The chatService.js file manages the logic for processing user requests and obtaining responses from the language model.

**Example:**

```javascript
const activityModel = require("../models/chatModels");

module.exports = {
  chatActivity: async (body) => {
    return await activityModel.chatActivity(body);
  },
};
```
